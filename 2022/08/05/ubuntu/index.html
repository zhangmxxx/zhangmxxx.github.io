Clash for Ubuntu

Guide for lauching clash through terminal on ubuntu.

1. Where to find clash?

​ root/usr/bin or just whereis clash

2. How to turn on VPN?

​ run clash and turn on Proxy(manual mode)

3. What does it do when launched?

​ It will try to find the host through Proxy setting(so you should turn manual mode on), and “occupy the host”. It keeps going until you kill the process.

​ From above, when using common connection, Proxy setting must be turned off.

4. How to terminate?

​ ctrl-C or sudo kill -9 PID.

​ You can get PID which is occupying the host by sudo lsof -i tcp:HostID (host id fits like 192.168.1.1: HostId)

5. Run Clash with the terminal off

​ nohup clash & This command will also launch a process (and can occupy the host when it has been occupied by the common process above, which will lead to error if another common attempt is made). It will append the log to a file under ~. To terminate clash by this way, killing is the only way.
